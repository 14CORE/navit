const mbrolib_voice** mbrolib_list_voices ( void *  the_handle)

Returns the list of the installed mbrola databases. The databases are searched according to the MBROLA_PATH environment variable if set, or under a default path otherwise (see MBROLA_PATH in mbrolib.c).

An array of voices is returned. The last item is set to NULL. The caller must not free the returned items or the array.

Parameters
the_handlepreviously given by mbrolib_init.
Returns
An array of voices.
